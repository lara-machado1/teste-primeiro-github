# Resumo sobre HTML

## O que é HTML?

HTML significa HyperText Markup Language (Linguagem de Marcação de Hipertexto).

Ele não é uma linguagem de programação.  
Ele é uma linguagem de marcação, ou seja, serve para marcar e organizar o conteúdo de um site.

O HTML é basicamente a estrutura da página.  
Ele organiza textos, títulos, imagens, links e outros elementos para o navegador entender o que mostrar na tela.

---

## Estrutura básica de um documento HTML

Todo arquivo HTML começa com uma estrutura padrão:

<!DOCTYPE html>  
<html>  
<head>  
<body>  

- `<!DOCTYPE html>` → Diz que o documento é HTML5.  
- `<html>` → Envolve todo o conteúdo da página.  
- `<head>` → Parte das configurações (como o título da aba do navegador).  
- `<body>` → Parte que aparece na tela para o usuário.

Tudo que a gente vê no site fica dentro do `<body>`.

---

## O que são tags?

As tags são os comandos do HTML.

Geralmente elas têm:
- Abertura
- Conteúdo
- Fechamento

Exemplo:

`<p>Olá mundo</p>`

- `<p>` → abre a tag  
- Olá mundo → conteúdo  
- `</p>` → fecha a tag  

A maioria das tags funciona assim.

---

## Principais tags que vimos

### `<h1>` até `<h6>`
São usadas para títulos.  
`<h1>` é o mais importante e maior.  
`<h6>` é o menor.

---

### `<p>`
Serve para criar parágrafos.

---

### `<a>`
Serve para criar links.  
Ela usa o atributo `href` para dizer para onde o link vai levar.

---

### `<img>`
Serve para colocar imagens na página.  
Ela usa o atributo `src` para indicar o caminho da imagem.

---

### `<div>`
A `<div>` serve para dividir e organizar partes do site.

Ela funciona como uma “caixa” que agrupa elementos.  
Ajuda muito na organização do código e também no alinhamento quando usamos CSS.

---

## Conclusão

O HTML é a base de qualquer site.  
Ele organiza a estrutura da página e diz para o navegador o que deve aparecer na tela.

Sem HTML, não existe site.
